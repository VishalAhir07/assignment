# WayGood

This is my submission for the WayGood React Native intern assignment

### About

I built a simple mobile app using React Native and Expo that shows a list of study abroad universities. You can tap on any university to see more details about it like tuition fees, duration and language.

### Screens

- Home Screen - shows list of universities
- Details Screen - shows full info about selected university

### Tech Used

- React Native with Expo
- TypeScript
- React Navigation
